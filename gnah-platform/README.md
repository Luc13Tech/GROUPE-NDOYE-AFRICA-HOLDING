## 🚀 DÉPLOIEMENT

### 1. Pousser sur GitHub
```bash
git init
git add .
git commit -m "GNAH Platform v1.0 - Complete"
git remote add origin https://github.com/VOTRE_USERNAME/gnah-platform.git
git push -u origin main
```

### 2. Déployer sur Vercel
1. Aller sur vercel.com → New Project
2. Importer le repo GitHub
3. Build Command: `npm run build`
4. Output Directory: `build`
5. Cliquer Deploy

### 3. Accès Admin
- URL: `votre-site.vercel.app/admin`
- Mot de passe: `admin123`
